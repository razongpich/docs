## Installing Redis
1. Ensure that your target host is exist in inventory
2. Inside the playbooks, specify host your wanna install redis on.

<ResponseField name="redis_password" type="string" required>
  Set redis password to require pass.
</ResponseField>
```bash
ansible-playbook playbooks/redis.yaml -i inventory.yaml
```

## Redis Server

<ResponseField name="name" type="string" required>
  Name of server with convetion of **environment-server-sequence** e.g.
  `prod-api-01` `dev-listener-02`
</ResponseField>

```bash
ansible-playbook playbooks/redis_server.yaml --extra-vars="name=prod-redis-01"
```
